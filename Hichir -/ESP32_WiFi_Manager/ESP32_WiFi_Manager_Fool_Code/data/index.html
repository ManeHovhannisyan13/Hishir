🔗 Բացել ամբողջական ձեռնարկը

Wi‑Fi կարգավորում, WEB միջավայր և դեղերի հիշեցումների ավելացում

Նախքան սկսելը
  • Սարքը միացրեք հոսանքին և պահեք հեռախոսը կամ համակարգիչը սարքին մոտ։
  • Պատրաստեք տան Wi‑Fi-ի անունը, գաղտնաբառը և Telegram Chat ID-ն։
  • Եթե էջը չի բացվում, ստուգեք, որ միացված եք ճիշտ Wi‑Fi ցանցին։
Առաջին միացում և Wi‑Fi կարգավորում

1. Միացրեք սարքը հոսանքին։ Սպասեք մի քանի վայրկյան, մինչև գործարկվի։ Առաջին կարգավորման ժամանակ սարքը ստեղծում է «ՀԻՇԻՐ» Wi‑Fi ցանցը։

2. Wi‑Fi ցանկում ընտրեք «ՀԻՇԻՐ» ցանցը։ Browser-ում գրեք՝ 192.168.4.1

3. Լրացրեք տան Wi‑Fi-ի անունը, գաղտնաբառը և Telegram Chat ID-ն, սեղմեք «Պահպանել»։

Կարևոր՝ Պահպանումից հետո նորից միացեք տան Wi‑Fi-ին։ Եթե WEB էջը չի բացվում, նայեք սարքի էկրանին ցուցադրված IP հասցեն։

WEB միջավայրում մուտք և դեղի ավելացում

4. Գրանցվեք WEB միջավայրում (մուտքանուն և գաղտնաբառ), սեղմեք «Մուտք»։

5. Սեղմեք «+» նշանը, ընտրեք տուփի համարը և լրացրեք դեղի տվյալները։

6. Գրեք դեղի անունը, քանակը, ընդունման շաբաթվա օրը և ժամը։ Սեղմեք «Լցնել դեղատուփը», ապա «Լցրեցի բոլորը» և «Հաստատել»։

Հիշեցման պահին
  • Նշված ժամին էկրանին երևում է դեղի տվյալը, միանում են ձայնային և լուսային ազդանշանները։
  • Telegram-ում ուղարկվում է ծանուցում։
  • Դեղը վերցնելուց հետո սեղմեք սարքի «*» կոճակը՝ buzzer-ը անջատելու համար։
Արագ խնդիրների լուծում
  • WEB էջը չի բացվում՝ ստուգեք Wi‑Fi կապը և IP հասցեն։
  • Telegram ծանուցում չկա՝ ստուգեք Chat ID-ն և ինտերնետ կապը։
  • Դեղի տվյալները չեն պահպանվել՝ կրկին սեղմեք «Հաստատել» և թարմացրեք էջը։

Wi‑Fi setup, web interface and adding medicine reminders

Before you start
  • Plug in the device and keep your phone or computer nearby.
  • Prepare your home Wi‑Fi name, password and Telegram Chat ID.
  • If the page does not open, check that you are on the correct Wi‑Fi network.
First setup and Wi‑Fi

1. Power on the device and wait a few seconds. On first setup it creates the «HISHIR» Wi‑Fi network.

2. Connect to «HISHIR» and open 192.168.4.1 in your browser.

3. Enter home Wi‑Fi credentials and Telegram Chat ID, then press Save.

Important: Reconnect to your home Wi‑Fi after saving. If the web page fails, check the IP on the device screen.

Web login and adding medicine

4. Register and log in on the web page.

5. Press «+», choose a box number and fill in medicine details.

6. Enter name, quantity, weekday and time. Use Fill box, then I filled all, then Confirm.

At reminder time
  • The screen shows the medicine; sound and light alerts activate.
  • A Telegram notification is sent.
  • Press «*» on the device after taking the medicine to stop the buzzer.
Quick troubleshooting
  • Web page won't open — check Wi‑Fi and IP address.
  • No Telegram message — check Chat ID and internet.
  • Data not saved — press Confirm again and refresh.
